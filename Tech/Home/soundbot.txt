====== soundbot ======
Created Saturday 27 January 2018

====== minimalist modular manipulation of sound ======

* ~~based on fedora 28~~ ?? using latest stable version of rt kernel
* system tasks managed with ansible
* lowprint ui, using i3 window manager
* ~~plugins compiled with maths that lessen cpu usage~~
* session management with nsm, midilings, bash-functions and aliases
* various helpful scripts
* designed to produce audio in a live or studio setting
